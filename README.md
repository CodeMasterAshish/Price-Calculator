# Price Calculator

A simple Price Calculator built using HTML, CSS, and JavaScript.

## Features

- Enter product price
- Enter quantity
- Calculate total cost instantly
- Clean and responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How It Works

1. Enter the product price.
2. Enter the quantity.
3. Click the **Calculate** button.
4. The total price will be displayed.

## Formula

Total Price = Product Price × Quantity
